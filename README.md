# Scripture Converter

Scripture Converter is a CLI (to be implemented) that detects, analyzes, and converts between different digital Bible formats.

## Formats

Formats to be supported (this list may be changed):

JSON:

- Scripture Burrito  JSON
- Open Scriptures JSON
- Bible API JSON formats (As used by YouVersion, Crossway, ...)

XML:

- USX (Unified Scripture XML)
- OSIS (Open Scripture Information Standard)
- Zefanja XML
- SWORD XML aka. OSIS modules
